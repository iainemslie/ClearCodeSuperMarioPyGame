## Environment Setup

```sh
$ python -m venv pirateenv
$ source ./pirateenv/Scripts/activate
$ pip install requirements.txt
```

## Pygame Note

`pygame-ce` is used instead of `pygame` since it has

- frequent updates
- faster runtime
- backwards-compatibility
